# PolyglotVision

Real-time video call with live translation and AI-driven lip-sync.  
Break language barriers instantly – all in your browser.

## How it works
- WebRTC video call via PeerJS
- Speech recognition (Web Speech API)
- Neural machine translation (LibreTranslate via serverless function)
- Text-to-speech (Speech Synthesis)
- Animated mouth overlay for lip-sync

## Tech Stack
- Next.js (React)
- PeerJS (WebRTC)
- Tailwind CSS
- Vercel (deployment)
